Interface - Gestion de fichiers

Afin de s'y retrouver dans les fichiers de notre jeu, qui vont devenir nombreux très vite, il faut impérativement penser à l'organisation de notre dossier de projet en amont. Cela peut être fait depuis Godot ou depuis l'explorateur de fichiers de l'ordinateur. Pour accéder rapidement à l'explorateur de fichiers depuis Godot, il suffit de faire un clic droit dans le panneau Système de fichiers et cliquer sur Ouvrir dans le gestionnaire de fichiers.

Dans le dossier de votre jeu, vous trouverez plus de fichiers que ceux affichés dans Godot. C'est normal, Godot crée des fichiers d'informations qui ne sont utiles qu'au moteur de jeu et donc qui n'ont pas lieu d'apparaître dans les ressources utilisables depuis l'éditeur.

Une chose à remarquer tout de même, les fichiers images, son ou autre sont très souvent suivis d'un fichier portant le même nom mais dont l'extension est .import, ce fichier contient les informations d'importation que Godot a générées. Il faut absolument le déplacer avec le fichier source si tu souhaites le changer de place. Sinon, cela risque de casser des liens entre les données dans tes différentes scènes.

Comme tu peux le voir, j'ai créé 5 nouveaux dossiers. C'est une organisation classique. On retrouve :

  • Un dossier Préfabs, raccourci de préfabriqué. Il contiendra tous les objets réutilisables de ton jeu (Un monstre, un objet, etc...)
  • Un dossier Scènes qui contiendra toutes les scènes principales de ton jeu (Menu, Mondes, etc...)
  • Un dossier Scripts qui contiendra tous les scripts généraux ou non assignés à un préfabriqué.
  • Un dossier Sons pour tous les sons.
  • Un dossier Textures pour toutes les images.

Bien sûr, il n'y a pas d'obligation, mais c'est la base qui sera utilisée par la suite.
