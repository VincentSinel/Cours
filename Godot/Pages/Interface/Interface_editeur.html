Interface - Interface de l'éditeur

Organisation

Voici l'éditeur lorsque l'on lance le projet pour la première fois. Celui-ci est complexe au premier abord mais on peut découper les différentes informations.

Voici le découpage principale avec les élèments les plus utiles pour démarrer un projet :

  1. Le panneau Scène contient la liste de tous les objets actuellement présent dans notre scène (la vue centrale). Il permet d'organiser, selectionner et hiérarchiser les différents élèments de notre jeu.
  2. Le panneau Système de fichiers est un explorateur de fichier qui montre ce que l'on trouve dans le dossier de sauvegarde du projet.
  3. Le panneau Inspecteur affiche les informations et propriétés d'un objet lorsque celui-ci est selectionné.
  4. Le panneau centrale montre une prévisualisation de la scène selectionné. Par default, la vision 3D est affiché mais on peut basculer sur la vision 2D dans le menu centrale en haut de l'écran.
  5. La barre des menus donne accés à des paramètres globaux de l'éditeur ou du projet.
  6. Les panneaux en bas de l'écran son des menus de paramètrages supplémentaire lorsque des objets complexe sont selectionné (Tilemaps, Animations, Audio, ...). Les menus peuvent changer en fonction de l'objet selectionné. Le menu Sortie indique toutes les actions effectué dans l'éditeur ainsi que les retours console que nécessaire au débuggage du jeu. Il permet aussi d'avoir des informations sur les éventuelles bug de ton jeu.
  7. Le panneau Signaux indique la liste des signaux que peut renvoyer un objet en fonction de différentes actions du jeu. Il permet de lié ses actions à un script et ainsi executer du code lorsque certain évenement ont lieux.
  8. Les bouttons en haut à droite permettent de Lancer le programme, de le mettre en pause ou de lancer une scene particulière.
Les deux panneaux bleus restant ainsi que les onglets cachés, proposent des outils non nécessaire pour débuter sur le logiciel.

Premiers noeuds

Les tutoriels de ce site se limite à la création de projet en 2 dimensions. Pour commencer et avoir la vue classique, il faut selectionner une scéne 2D dans le panneau des Scène.

Dans le panneau Scène, on voit désormais apparaitre notre premier noeud que l'on appelle le Noeud racine. Celui-ci etant directement selectionné, l'Inspecteur affiche ses propriétés.

On peut désormais commencer à créer des objets. Par exemple, on peut ajouter une image à la scene en effectuant un glisser déposer de l'image présente dans le Système de fichiers vers la prévisualisation de la scène.

L'image viens s'afficher sur la parti centrale et un noeud est automatiquement créer sous notre Noeud racine. On peut alors modifier les propriétés de ce noeud pour le modifier. Par exemple effectuer une rotation. On peut noter que ce noeud est de type Sprite2D

A noter que lorsque l'on dézoome sur la prévisualisation de la scène, on voit apparaitre le rectangle suivant :

Ce rectangle délimite la zone visible actuel. Si vous lancer le jeu vous ne verrez que ce qui se trouve à l'intérieur de ce rectangle car rien n'a encore été modifié.

Pour te déplacer facilement sur la prévisualisation de la scène, il est préférable de garder le mode selection (boutton en forme de curseur le plus à gauche en dans la barre d'outils). On peut alors selectionner les objets avec un clic gauche et déplacer la camera avec le clic de la molette.
