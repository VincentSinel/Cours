Interface - Interface de l'éditeur

Organisation

Voici l'éditeur lorsque l'on lance le projet pour la première fois. Celui-ci est complexe au premier abord mais on peut découper les différentes informations.

Voici le découpage principal avec les éléments les plus utiles pour démarrer un projet :

  1. Le panneau Scène contient la liste de tous les objets actuellement présents dans notre scène (la vue centrale). Il permet d'organiser, sélectionner et hiérarchiser les différents éléments de notre jeu.
  2. Le panneau Système de fichiers est un explorateur de fichiers qui montre ce que l'on trouve dans le dossier de sauvegarde du projet.
  3. Le panneau Inspecteur affiche les informations et propriétés d'un objet lorsque celui-ci est sélectionné.
  4. Le panneau central montre une prévisualisation de la scène sélectionnée. Par défaut, la vision 3D est affichée mais on peut basculer sur la vision 2D dans le menu central en haut de l'écran.
  5. La barre des menus donne accès à des paramètres globaux de l'éditeur ou du projet.
  6. Les panneaux en bas de l'écran sont des menus de paramétrages supplémentaires lorsque des objets complexes sont sélectionnés (Tilemaps, Animations, Audio, ...). Les menus peuvent changer en fonction de l'objet sélectionné. Le menu Sortie indique toutes les actions effectuées dans l'éditeur ainsi que les retours console nécessaires au débogage du jeu. Il permet aussi d'avoir des informations sur les éventuelles bugs de ton jeu.
  7. Le panneau Signaux indique la liste des signaux que peut renvoyer un objet en fonction de différentes actions du jeu. Il permet de lier ses actions à un script et ainsi exécuter du code lorsque certains événements ont lieu.
  8. Les boutons en haut à droite permettent de Lancer le programme, de le mettre en pause ou de lancer une scène particulière.
Les deux panneaux bleus restants ainsi que les onglets cachés proposent des outils non nécessaires pour débuter sur le logiciel.

Premiers noeuds

Les tutoriels de ce site se limitent à la création de projets en 2 dimensions. Pour commencer et avoir la vue classique, il faut sélectionner une scène 2D dans le panneau des Scènes.

Dans le panneau Scène, on voit désormais apparaître notre premier noeud que l'on appelle le Noeud racine. Celui-ci étant directement sélectionné, l'Inspecteur affiche ses propriétés.

On peut désormais commencer à créer des objets. Par exemple, on peut ajouter une image à la scène en effectuant un glisser-déposer de l'image présente dans le Système de fichiers vers la prévisualisation de la scène.

L'image vient s'afficher sur la partie centrale et un noeud est automatiquement créé sous notre Noeud racine. On peut alors modifier les propriétés de ce noeud pour le modifier. Par exemple, effectuer une rotation. On peut noter que ce noeud est de type Sprite2D

À noter que lorsque l'on dézoome sur la prévisualisation de la scène, on voit apparaître le rectangle suivant :

Ce rectangle délimite la zone visible actuelle. Si vous lancez le jeu, vous ne verrez que ce qui se trouve à l'intérieur de ce rectangle car rien n'a encore été modifié.

Pour te déplacer facilement sur la prévisualisation de la scène, il est préférable de garder le mode selection (boutton en forme de curseur le plus à gauche en dans la barre d'outils). On peut alors sélectionner les objets avec un clic gauche et déplacer la caméra avec le clic de la molette.
