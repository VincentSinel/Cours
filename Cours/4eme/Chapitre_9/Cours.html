Egalité d'expressions littérales (rappels) :

Définition :

Deux expressions littérales sont égales si elles prennent toujours la même valeur quand on remplace les variables par des valeurs numériques.

Exemples :

• $4x+5=2x+5+2x$
• $2l+2L=2(l+L)$
• $x^4=x\times x\times x\times x$

Réduction :

Réduire une expression littérale, c’est l’écrire avec le moins de termes possibles

Distributivité :

Définition :

La distributivité $k\times(m+n)=k\times m+k\times n$ permet d'écrire un produit entre un nombre et une somme comme une somme de deux produits.
$k$ $\times($ $m+n$ $)$ = $\textcolor{#C00000}{k}\times \textcolor{#0070C0}{m}$ $+$ $\textcolor{#C00000}{k}\times \textcolor{#0070C0}{n}$
nombre $\times$ somme produit $+$ produit
produit somme

Remarque :

La distributivité permet de passer d'une somme à un produit et inversement.
On l'utilise de manière logique dans les calculs comme : $101\times42 = 100\times42 + 1\times42 = 4200 + 42 = 4242$

Développement :

Lorsque l'on passe d'un produit à une somme, on développe
$\textcolor{#C00000}{k}\times(\textcolor{#0070C0}{m+n})$ $k$ = $\textcolor{#C00000}{k}\times \textcolor{#0070C0}{m}+ \textcolor{#C00000}{k}\times \textcolor{#0070C0}{n}$
Développement

Exemples :

On multiple chaque terme dans la parenthèse par le facteur se trouvant devant.

$\textcolor{#ED1C24}{2x}(3x+5)$ $= \textcolor{#ED1C24}{2x}\times3x + \textcolor{#ED1C24}{2x}\times5$ $= 6x^2+10x$

$\textcolor{#ED1C24}{-3x}(-2x+7)$ $= \textcolor{#ED1C24}{-3x}\times(-2x) + \textcolor{#ED1C24}{(-3x)}\times7 $ $= 6x^2-21x$

$\textcolor{#ED1C24}{-2x}(-8x^2+5x -4)$ $= \textcolor{#ED1C24}{-2x}\times(-8x^2) + \textcolor{#ED1C24}{(-2x)}\times5x - \textcolor{#ED1C24}{(-2x)}\times4 $ $= 16x^3-10x^2+8x$

Remarque :

Le cas de figure suivant peut se présenter, on prend alors l'opposé de chacun des termes entre parenthèse :

  • $-(3x-7) = -3x + 7$
  • $-(-5x^2+4x-1,5) = 5x^2 - 4x+1,5$

Factorisation :

Lorsque l'on passe d'une somme à un produit, on factorise
$\textcolor{#C00000}{k}\times(\textcolor{#0070C0}{m+n})$ $k$ = $\textcolor{#C00000}{k}\times \textcolor{#0070C0}{m}+ \textcolor{#C00000}{k}\times \textcolor{#0070C0}{n}$
Factorisation

Exemples :

  • $8x+12 = \textcolor{#ED1C24}{4}\times2x + \textcolor{#ED1C24}{4}\times3 = \textcolor{#ED1C24}{4}(2x+3)$
  • $3x^2+5x = \textcolor{#ED1C24}{x}\times3x + \textcolor{#ED1C24}{x}\times5 = \textcolor{#ED1C24}{x}(3x+5)$
  • $-24x^2-12x = \textcolor{#ED1C24}{-12x}\times2x + \textcolor{#ED1C24}{(-12x)}\times1 = \textcolor{#ED1C24}{-12x}(2x+1)$

Remarque :

Il existe une infinité de factorisation possible car on peut factoriser par n'importe quel nombre.

Prenons $24x + 16$, il est possible de le factoriser comme suis :

  • $\textcolor{#ED1C24}{8}(3x+2)$
  • $\textcolor{#ED1C24}{4}(6x+4)$
  • $\textcolor{#ED1C24}{2}(12x+8)$
  • $\textcolor{#ED1C24}{5}(4,8x+3,2)$
  • $\textcolor{#ED1C24}{7}(\frac{24}{7}x+\frac{16}{7})$
  • $\textcolor{#ED1C24}{23}(\frac{24}{23}x+\frac{16}{23})$
  • $\textcolor{#ED1C24}{16}(\frac{3}{2}x+1)$
